Mi Biblioteca

Esta es una aplicación móvil desarrollada a través de Kotlin y el IDE de Android Studio, realizando la autenticación y gestión de bases de datos a través de Firebase con el modelo CRUD y el uso de API para acceder a la ubicación del usuario. Su función es realizar un seguimiento detallado de los libros que el usuario ha leído, prestado o reservado y para acceder a la información general de las bibliotecas más cercanas a su ubicación actual; lograr un acceso amigable y un uso adecuado de los datos y archivos multimedia por parte del usuario.
